# Purpose
Run a final QA pass on the current version before publish or handoff.

# Checklist
## Functional
- Navbar links scroll to correct sections
- Mobile menu works
- Filter tabs work
- Contact buttons work
- Video heroes load correctly
- Modals or interactive cards open and close correctly

## Asset integrity
- No broken image paths
- No broken video paths
- Favicon loads
- Logo path is correct

## UX
- No text overlaps
- Good contrast on dark background
- Cards behave correctly on desktop and mobile
- Final CTA is visible and readable

## Production readiness
- Title and description present
- Social and OG basics if present
- File names consistent
- No leftover placeholders
- No duplicate dead sections

# Output format
- Fixed now
- Still pending
- Publish-ready status
- Recommended next step
